# grpc-ms

Microservice architecture project with gRPC and protobuf.
written in golang.

*Stack used* - docker, go-micro, Makefile, docker-compose, mongodb, circleci, gcloud & kubernetes.  

### Microservices
Container management system,
-   consignments
-   inventory
-   users
-   authentication
-   roles
-   vessels
